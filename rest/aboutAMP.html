Thank you for checking out AMP Stats, our RESTful API project. This was the result of the collaborative efforts of Zachary Arrasmith, Brian Mize, and Tim Perkins. The project name came from the first letter of our last names, and we thought that AMP was an appropriate name for a project that deals with athletics.

For this project, we were to create a REST server that would allow for both the retrieval and the maintenance of data. For this we provided two end points; one for users who will just be retrieving data, and the other for admins who will be maintaining the data.

Users are only given access to GET methods that retrieve the recourses so that they can be displayed however the user sees fit. Admins are given all the User GET methods along with methods for POST, PUT, and DELETE.

We have created REST styled documentation for Users that can be found by clicking on API Documents. We also have REST styled documentation for Admins that can be access by clicking on Login. Admin access requires a user name and password.

We have also created a front-end UI for casual users that uses our User API to be able to view our EWU Football stats data set.

Feel free to contact us with any questions, comments or concerns.
